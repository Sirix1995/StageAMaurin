This is the API documentation for the GroIMP software, which is also available for download as a single archive at the download page of grogra.de.

RGG users might want to have a look at the following packages:

  • {@link de.grogra.rgg} defines the basic classes for relational growth grammars.
  • {@link de.grogra.turtle} contains turtle commands and related classes for standard L-system modelling.
  • {@link de.grogra.imp3d.objects} contains a lot of 3D classes which can be used in relational growth grammars.
  • {@link de.grogra.math} defines, among others, classes for spline surfaces and curves.
